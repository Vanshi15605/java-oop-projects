# Bank Management System

This project is a simple implementation of a banking system using Java.

It demonstrates how Object-Oriented Programming concepts can be applied to design real-world systems.

## Features

- Deposit and withdrawal operations
- Savings account with interest calculation
- Current account with overdraft facility
- Input validation for invalid transactions

## Concepts Used

- Encapsulation for data protection
- Inheritance to create specialized account types
- Method overriding for customized withdrawal behavior

## How It Works

The system consists of a base class `BankAccount` and two derived classes:
- `SavingsAccount`
- `CurrentAccount`

The `Main` class demonstrates how these accounts behave with different operations.

## Learning Outcome

This project helped in understanding how to structure code properly and apply OOP concepts in a practical scenario.
